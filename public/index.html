Commands
Slack commands
  • /list {typeOfFile}
    • typeOfFile: Optional parameter. By default: all. For multiple values use coma separated values Ex.images,spaces. Posible values:
      • all : All files
      • spaces : Posts
      • snippets : Snippets
      • images : Image files
      • gdocs : Google docs
      • zips : Zip files
      • pdfs : PDF files
  • /how_use_the_bot: Shows the help of the bot.underconstruction
